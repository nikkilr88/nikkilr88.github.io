Twitter Bot

Node API Heroku

This bot tweets articles and tracks / retweets popular developer hashtags. It connects to the YouTube API to search and post coding tutorials from selected channels.

To avoid retweeting spam posts, it counts hashtags and only retweets posts with four or less.

Switchr

React API Parcel

Switchr is a Chrome extension that toggles other extensions on and off with just a click. Extensions are sorted by enabled status, and there is a popup search to easily find what you're looking for.
